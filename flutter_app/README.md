# Application Flutter - Interface Graphique

Une application Flutter moderne avec une interface graphique élégante.

## Fonctionnalités

- **Tableau de bord** : Vue d'ensemble avec statistiques et activités récentes
- **Profil utilisateur** : Gestion du profil avec options personnalisables
- **Paramètres** : Configuration de l'application avec différents paramètres

## Installation

1. Assurez-vous d'avoir Flutter installé sur votre système
2. Naviguez vers le dossier du projet :
   ```bash
   cd flutter_app
   ```
3. Installez les dépendances :
   ```bash
   flutter pub get
   ```
4. Lancez l'application :
   ```bash
   flutter run
   ```

## Structure du projet

```
flutter_app/
├── lib/
│   └── main.dart          # Fichier principal de l'application
├── pubspec.yaml           # Dépendances et configuration
└── README.md              # Documentation
```

## Interface

L'application comprend trois écrans principaux accessibles via une barre de navigation en bas :

1. **Accueil** : Tableau de bord avec statistiques et activités
2. **Profil** : Informations utilisateur et options
3. **Paramètres** : Configuration de l'application

## Technologies utilisées

- Flutter 3.0+
- Material Design 3
- Navigation Bar moderne
